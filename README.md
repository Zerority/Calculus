# Visualiseur de Calcul

## Version 2.0

### Introduction

Le visualisur de calcul est un projet personnel développé en Python pour faciliter l'apprentissage des notions fondamentales du calcul, notamment :

* Les dérivées ;
* Les intégrales ;
* Les limites.

L'objectif de ce projet est d'aider les utilisateurs à mieux comprendre ces concepts grâce à leur représentation graphique, et aussi calculer des résultats mathématiques

Ce projet a été réalisé dans le cadre de mon auto-apprentissage de Python ainsi que des bibliothèques SymPy, NumPy, Matplotlib et Tkinter.

---

## Fonctionnalités

### 1. Dérivées

L'utilisateur saisit :

* Une fonction mathématique ;
* La valeur de (x_0), correspondant au point de tangence.

Le programme effectue ensuite les opérations suivantes :

* Calcul de la dérivée de la fonction ;
* Tracé du graphique de la fonction ;
* Calcul et tracé de la tangente au point choisi.

---

### 2. Intégrales

L'utilisateur saisit :

* Une fonction mathématique ;
* La borne inférieure ;
* La borne supérieure.

Le programme effectue ensuite les opérations suivantes :

* Calcul de l'intégrale définie sur l'intervalle ([a,b]) ;
* Tracé du graphique de la fonction ;
* Coloration de la région correspondant à l'aire calculée.

---

### 3. Limites

L'utilisateur saisit :

* Une fonction mathématique ;
* Le point vers lequel la variable tend.

Le programme effectue ensuite les opérations suivantes :

* Calcul de la limite de la fonction en ce point ;
* Vérification de la valeur de la fonction au point considéré ;
* Tracé du graphique au voisinage du point étudié ;
* Représentation graphique de la limite.

---

## Technologies utilisées

### Python

Langage de programmation principal du projet.

### SymPy

Utilisé pour :

* Manipuler des expressions symboliques ;
* Calculer des dérivées ;
* Calculer des intégrales ;
* Calculer des limites.

### NumPy

Utilisé pour :

* Générer des ensembles de valeurs numériques ;
* Préparer les données nécessaires à la représentation graphique.

### Matplotlib

Utilisé pour :

* Représenter graphiquement des fonctions ;
* Afficher des tangentes ;
* Illustrer des intégrales ;
* Visualiser les notions de calcul différentiel et intégral.

### Tkinter

Utilisé pour :

* Concevoir l'interface graphique du programme ;
* Gérer les interactions avec l'utilisateur ;
* Afficher les résultats et les représentations graphiques de manière interactive.
